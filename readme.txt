รายละเอียดโครงงานมีดังนี้
1. System requirements
การทำงานของ server จำเป็นจะต้องติดตั้ง Node.js สามารถดาวน์โหลดได้ที่
https://nodejs.org/en/download/

2. Feature
เป็นเกมบนเว็บหลายผู้เล่น ซึ่งผู้เล่นแต่คนสามารถล็อกอินเข้ามาเพื่อเล่นแข่งขันกัน
กินอาหารเพื่อเก็บคะแนน และหลบหลีกมอนสเตอร์ที่กินแล้วจะลดคะแนน
โดยจะมีผู้เล่นเพียงคนเดียวที่จะถูกบันทึกคะแนนสูงสุด
หากผู้เล่นล็อกอินด้วยบัญชีเดิมจะสามารถเล่นต่อได้โดยไม่ถูกรีเซ็ตคะแนนและตำแหน่ง

3. How to play
	3.1 เปิดเซิร์ฟเวอร์ด้วยไฟล์ StartServer.bat
	3.2 ทำการเปิดเว็บบราวเซอร์(Google Chrome recommended)
	3.3 เนื่องจากเซิร์ฟเวอร์ถูกตั้งค่าให้ทำงานที่พอร์ต 2000 * ให้เข้าไปที่
		localhost:2000
	หรือหากเป็นคอมพิวเตอร์, สมาร์ทโฟนซึ่งอยู่ในวงแลนเดียวกัน สามารถใช้ IP Address ของเครื่องที่ทำงานเป็นเซิร์เวอร์เพื่อเข้ามาเล่นเกมได้ 
	เช่น เซิร์ฟเวอร์มี IP Address เป็น 192.168.173.1 ให้เข้าไปที่
		192.168.173.1:2000
	3.4 ทำการล็อกอินเพื่อเข้าเล่นเกม **

* สามารถ edit ไฟล์ StartServer.bat เพื่อเปลี่ยนพอร์ตที่จะให้เซิร์ฟเวอร์ทำงานได้ เช่น ต้องการให้ทำงานที่พอร์ต 5555 จะแก้ไขไฟล์ได้ดังนี้
	node server.js 5555
** การล็อกอินครั้งแรกนั้นถือว่าเป็นการสร้างบัญชีอัตโนมัติ สามารถใช้ username และ password ใดๆก็ได้
